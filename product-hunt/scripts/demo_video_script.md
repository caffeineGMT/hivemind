# Demo Video Script (60 seconds)

## Video Specs
- **Length:** 60 seconds (Product Hunt optimal)
- **Format:** MP4, 1080p (1920×1080)
- **Aspect Ratio:** 16:9
- **File Size:** <100MB
- **Voiceover:** Optional (can use text overlays instead)
- **Music:** Upbeat, tech-focused background track (low volume)

---

## Script & Storyboard

**Visual:** Screen showing cluttered workspace with multiple ChatGPT tabs, terminal windows, scattered notes

**Text Overlay:** "Hivemind Engine does it all for you"
**Voiceover:** "Hivemind Engine automates everything."

1. Click "New Business"
2. Type: "Build an AI newsletter"
3. Click "Create"
**Text Overlay:** "Just describe your goal"
**Voiceover:** "Describe what you want to build..."

**Visual:** Screen recording of:
- Dashboard showing agents spawning in real-time
- Agent cards appearing: "Web Scraper", "Content Writer", "Email Sender"
- Status indicators turning green

**Visual:** Screen recording of:
- Click into task details

**Text Overlay:** "Monitor everything in real-time"
**Voiceover:** "Track tasks, costs, and logs in real-time."

### [45-52 seconds] Demo Part 4: Results

**Text Overlay:** "Deploy to production in one click"

- Hivemind logo animates in

  Line 1: "Hivemind Engine"

## Recording Checklist

### Pre-Recording
- [ ] Clear browser cache for clean state
- [ ] Set up demo data (sample newsletter project)
- [ ] Disable desktop notifications
- [ ] Hide browser bookmarks bar
- [ ] Set resolution to 1920×1080
- [ ] Test screen recording software (OBS, Loom, or QuickTime)
- [ ] Prepare cursor highlighting (optional)

### During Recording
- [ ] Move cursor slowly and deliberately
- [ ] Pause 2 seconds on each important state change
- [ ] Record multiple takes (choose best)
- [ ] Capture at 60fps for smooth playback

### Post-Production
- [ ] Trim to exactly 60 seconds
- [ ] Add text overlays (use simple, bold font)
- [ ] Add background music (royalty-free from Epidemic Sound or Artlist)
- [ ] Add subtle sound effects (clicks, success sounds)
- [ ] Color correct for consistency
- [ ] Export at 1080p, H.264 codec
- [ ] Test playback on multiple devices

## Voiceover Tips (If Recording)

### Voice Direction:
- Tone: Friendly, confident, slightly casual
- Pace: Moderate (not too fast, not too slow)
- Energy: Upbeat but not overly hyped
- Inflection: Emphasize key words ("autonomous", "one click", "real-time")

### Recording Setup:
- Use decent USB mic (Blue Yeti, Audio-Technica AT2020)
- Quiet room (minimize echo)
- Pop filter to reduce plosives
- Record multiple takes
- Edit out breaths, lip smacks, filler words

### Alternative: Text-to-Speech
If you don't want to record voiceover, use high-quality TTS:
- **ElevenLabs** (most natural-sounding)
- **Play.ht** (good alternative)
- **Murf.ai** (professional quality)

Voice recommendation: Male, neutral accent, 30s age, confident tone

## Text Overlay Style Guide

**Font:** Inter, SF Pro Display, or Montserrat (bold)
**Size:** 48-64px for main text, 32px for subtext
**Color:** White with subtle drop shadow (2px offset, 50% opacity)
**Animation:** Fade in + slight slide up (0.3s ease-out)
**Position:** Lower third or center (depending on scene)
**Background:** Optional dark gradient bar behind text for readability

## Pro Tips

1. **First 3 seconds are critical** – Hook viewers immediately or they'll scroll past
2. **Show, don't tell** – Screen recording > talking heads
3. **Keep it simple** – One idea per scene
4. **Mobile-friendly** – 70% of PH users are on mobile, test there first
5. **Silent playback** – Many watch without sound, ensure text overlays carry the message
6. **Fast pacing** – 60 seconds goes fast, every frame counts
7. **End with urgency** – "Limited offer", "Live today", "Try now"
