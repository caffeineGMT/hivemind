## Email Tool Recommendations

**Best for beta launches:**
- **Loops.so:** Simple, founder-friendly, great for small lists (<1,000)
- **Resend:** Developer-focused, React email templates, easy setup
- **Mailgun:** Programmatic sending, good for transactional + broadcast

**Avoid:**
- Mailchimp (overkill for small lists)
- SendGrid (too enterprise-y)

**Recommended setup:**
- Use Resend + React Email for clean, code-based templates
- Track clicks with UTM params (?utm_source=email&utm_campaign=ph_launch)
- Set up a simple dashboard to monitor open/click rates in real-time
