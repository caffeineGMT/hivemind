# Product Hunt Launch Checklist

Complete checklist for launching Hivemind Engine on Product Hunt.

---

## 📅 7 Days Before Launch

### Product Hunt Preparation
- [ ] Choose launch date (Tuesday recommended)
- [ ] Create Product Hunt account (if needed)
- [ ] Find Hunter with 1000+ followers
  - Check leaderboard: producthunt.com/leaderboard/hunters
  - Research recent successful hunts
  - Reach out with pitch (see hunter-search-guide.md)
- [ ] Create Product Hunt draft listing
- [ ] Add Hunter to draft (once agreed)

### Content Creation
- [ ] Write launch copy (tagline, description)
  - Tagline: < 60 characters
  - Description: 2-3 paragraphs
  - First comment: detailed intro
- [ ] Create demo video (60 seconds, < 100MB)
  - Script written
  - Screen recording captured
  - Voiceover added
  - Music added
  - Edited to 60 seconds
  - Exported as MP4
- [ ] Capture 5 screenshots (1920x1080)
  - Dashboard overview
  - Agent activity
  - Cost tracking
  - Deployed result
  - Mobile view
- [ ] Annotate screenshots
- [ ] Create cover image (1200x630px)
  - Design in Figma/Canva
  - Export as PNG
  - Compress to < 500KB
- [ ] Request testimonials from beta users
  - Email sent to all beta users
  - Follow up after 2 days
  - Collect at least 3 testimonials

### Landing Page
- [ ] Optimize landing page for PH traffic
- [ ] Add PH50 promo banner (visible, eye-catching)
- [ ] Test PH50 promo code (validates correctly)
- [ ] Add "Featured on Product Hunt" placeholder
- [ ] Ensure mobile-responsive
- [ ] Test on multiple browsers
- [ ] Fast load time (< 2 seconds)

### Analytics Setup
- [ ] PostHog installed and tested
- [ ] Plausible or GA4 configured
- [ ] UTM parameters added to all links
- [ ] Event tracking implemented
  - Signup
  - Promo code applied
  - Payment completed
  - Demo viewed
- [ ] Google Sheets dashboard created
- [ ] Real-time tracking tested

### Team Preparation
- [ ] Brief team on launch plan
- [ ] Assign roles (who upvotes when)
- [ ] Create upvote schedule
  - Wave 1: 12:00am-1:00am (10 people)
  - Wave 2: 9:00am-10:00am (15 people)
  - Wave 3: 6:00pm-7:00pm (10 people)
- [ ] Share Product Hunt link template
- [ ] Test upvote process with 2-3 people

## 🔥 Emergency Protocols

### If Ranking Drops
- [ ] Ask supporters for final push
- [ ] Post on more channels
- [ ] Offer limited-time bonus
- [ ] Engage more in comments

### If Negative Comments
- [ ] Respond within 15 minutes
- [ ] Acknowledge concern
- [ ] Offer to discuss privately
- [ ] Don't argue or get defensive

### If Technical Issues
- [ ] Post immediate comment: "Investigating [issue]"
- [ ] Fix ASAP
- [ ] Follow up: "Issue resolved"
- [ ] Offer credit to affected users

### If Hunter Goes MIA
- [ ] Self-hunt (you can still launch)
- [ ] Reach out to backup Hunter
- [ ] Ask Product Hunt support
