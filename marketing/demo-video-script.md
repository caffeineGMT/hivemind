# Hivemind in 60 Seconds - Demo Video Script

## 🎬 Recording Setup

**Duration:** Exactly 60 seconds
**Format:** 1920x1080 (1080p) screen capture
**Tool:** QuickTime Screen Recording (Mac) or OBS Studio
**Audio:** Optional voiceover OR on-screen text captions

---

## 📋 Shot-by-Shot Breakdown

### 0:00-0:05 - Hook (5 seconds)
**Screen:** Terminal with prompt ready
**Text Overlay:** "Watch me build a complete SaaS app... in 60 seconds"
**Action:** Cursor blinking at terminal prompt

### 0:05-0:10 - The Command (5 seconds)
**Screen:** Type command slowly and clearly
**Command:**
```bash
hivemind "Build a SaaS landing page with pricing, testimonials, and email capture"
```
**Text Overlay:** "One command. That's it."

### 0:10-0:20 - Agent Swarm Spawns (10 seconds)
**Screen:** Show tmux panes splitting
**Action:**
- Terminal splits into 4 panes
- Agent names appear: ProductManager, Designer, Frontend, Backend
- Show agents "thinking" with loading spinners
**Text Overlay:** "AI agents spawn and collaborate 24/7"

### 0:20-0:35 - Agents Working (15 seconds)
**Screen:** Quick cuts between agent outputs
**Show:**
- ProductManager: "Planning user flow..."
- Designer: "Creating hero section..."
- Frontend: "Building React components..."
- Backend: "Setting up API endpoints..."
**Text Overlay:** "Building, deploying, iterating—while you sleep"

### 0:35-0:45 - Deployment (10 seconds)
**Screen:** Terminal shows deployment log
**Output:**
```
✓ Build complete
✓ Deploying to Vercel...
✓ Live at https://my-saas-demo.vercel.app
```
**Text Overlay:** "Production deployment in minutes"

### 0:45-0:55 - Show Live Site (10 seconds)
**Screen:** Browser opens to deployed site
**Show:**
- Smooth scroll through landing page
- Hero section with CTA
- Pricing cards
- Testimonials
- Email capture form
**Text Overlay:** "Fully functional. Production-ready. No manual coding."

### 0:55-0:60 - CTA (5 seconds)
**Screen:** Final frame with branding
**Text:**
```
Try it free: hivemind.dev
Ship 10x faster with AI agents
```

## 🎥 Recording Checklist

- [ ] Clean terminal (clear history, remove clutter)
- [ ] Set terminal theme (dark theme, readable font size 14-16pt)
- [ ] Disable desktop notifications
- [ ] Close unnecessary apps
- [ ] Set browser to incognito mode (clean state)
- [ ] Have demo project ready to deploy
- [ ] Test full flow once before recording
- [ ] Record at 1080p, 30fps minimum
- [ ] Use text overlays (not voiceover) for better virality
- [ ] Export as MP4 (H.264 codec for Twitter compatibility)
