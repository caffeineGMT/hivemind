## 🎬 Video Production Guide

### 60-Second Structure
1. **0-5s:** Hook - "Watch me build a SaaS app in 60 seconds"
2. **5-10s:** Command - Type hivemind build command
3. **10-20s:** Agent spawn - Tmux splits into 4 panes
4. **20-35s:** Agents working - Quick outputs montage
5. **35-45s:** Deployment - Vercel deployment logs
6. **45-55s:** Live site - Browser scroll through landing page
7. **55-60s:** CTA - "Try it free: hivemind.dev"

### Technical Specs
- **Format:** MP4 (H.264 video, AAC audio)
- **Resolution:** 1920x1080 (16:9)
- **Frame rate:** 30fps or 60fps
- **File size:** Under 512MB (Twitter limit)
- **Captions:** On-screen text overlays
- **Music:** Optional upbeat background track

## 📊 Analytics & Tracking

### Implementation
1. **Twitter Pixel** - Install on website for conversion tracking
2. **Conversion Events** - Sign-up, purchase, page views
3. **UTM Parameters** - Unique tracking links for each source
4. **Plausible/PostHog** - Website analytics
5. **Custom Dashboard** - Google Sheets for daily metrics

### Key Metrics
- Video view breakdown (25%, 50%, 75%, 100%)
- Engagement rate (likes, RTs, replies)
- Click-through rate (CTR)
- Conversion rate (clicks → sign-ups)
- Cost per acquisition (CPA)
- Return on ad spend (ROAS)

### Daily Monitoring (10AM PT)
- Total spend vs budget
- CPV and CPC metrics
- Engagement and conversion rates
- Top performing content
- Adjust bids and targeting

## 💡 Key Decisions Made

### Strategic
1. **60 seconds video** - Optimal balance of detail and attention span
2. **Text overlays** - 80% watch muted, better virality
3. **$100 budget** - Low-risk validation test
4. **Week 1 focus** - Quick feedback loop for iteration

### Tactical
1. **Front-load budget** - Days 1-2 get more spend for visibility
2. **A/B test 3 variations** - Find winning message quickly
3. **Multi-touch attribution** - Track full user journey
4. **Promo code** - Incentivize early adoption

### Technical
1. **Twitter Pixel** - Essential for conversion tracking
2. **Plausible analytics** - Privacy-focused, simpler than GA
3. **Custom spreadsheet** - Centralized daily metrics
4. **UTM parameters** - Source attribution clarity
