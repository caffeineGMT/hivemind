# Marketing Infrastructure

This directory contains the marketing and user acquisition systems for Hivemind Engine.

## Files

- **seo.js** - SEO optimization (meta tags, structured data, sitemaps)
- **analytics.js** - Analytics integration (GA4, Mixpanel, tracking)
- **affiliates.js** - Affiliate program management
- **email-templates.js** - Email marketing automation
- **content-generator.js** - AI content generation
- **cmo-agent-tasks.md** - Detailed CMO task breakdown
